GeneWiseAutoencoderBasedASDClassifier.py: This code uses AutoEncoder to find the latent representation for each gene for an individual, averages it and then uses these numbers to classify whether that individual has ASD or not. 1 individual has N genes = N representations per individual.

SingleRepresentationAutoencoderBasedASD.py: This code uses AutoEncoder to find the latent representation for each individual, averages it and then uses these numbers to classify whether that individual has ASD or not. 1 representation for 1 individual.
